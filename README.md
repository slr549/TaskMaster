# TaskMaster - Sistem Manajemen Tugas Kolaborasi

Sebuah aplikasi manajemen tugas sederhana namun efektif yang dibangun untuk memfasilitasi kolaborasi antara **Slrahanzhar549** dan **Hanshanuraazma**. Proyek ini fokus pada kemudahan penggunaan dan antarmuka yang bersih menggunakan teknologi web dasar.

---

## 📋 Deskripsi Proyek
TaskMaster dirancang untuk membantu pengguna mencatat, memantau, dan menyelesaikan daftar tugas (to-do list) secara efisien. Proyek ini merupakan hasil kolaborasi teknis untuk mengimplementasikan logika JavaScript pada antarmuka web yang responsif.

## ✨ Fitur Utama
* **Manajemen Tugas:** Menambah tugas baru, menandai tugas yang selesai, dan menghapus tugas yang sudah tidak diperlukan.
* **Prioritas & Kategori:** Pengelompokan tugas berdasarkan tingkat urgensi.
* **Penyimpanan Lokal (Local Storage):** Data tugas tetap tersimpan di browser meskipun halaman di-refresh.
* **Desain Responsif:** Tampilan yang optimal baik di perangkat desktop maupun mobile.

## 🛠️ Teknologi yang Digunakan
Proyek ini sepenuhnya dibangun menggunakan teknologi *native* web:
* **HTML5:** Untuk struktur semantik aplikasi.
* **CSS3:** Untuk styling, tata letak Flexbox/Grid, dan animasi transisi.
* **Vanilla JavaScript:** Untuk logika manipulasi DOM, manajemen data, dan interaktivitas.
